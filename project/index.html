Esse é o nosso projeto da cadeira Programação Web

Este projeto aplica habilidades em JavaScript, HTML, CSS e consumo de APIs REST, utilizando o Back4App como back-end. Com dados do Censo Escolar 2023 e um recorte baseado no MiniMundo do Projeto de Banco de Dados (que visa exibir dados sobre bibliotecas nas escolas do estado de Pernambuco.) O dashboard foi criado como uma solução integrada para consolidar os trabalhos e colocar em prática os conhecimentos adquiridos.
